# HOJAI Automation Platform Vision — Complete Strategic Document

> **Created:** June 30, 2026
> **Source:** Strategic planning session
> **Status:** For audit and gap analysis

---

## TL;DR Summary

HOJAI should NOT copy n8n workflows. The strategic positioning is:
- **n8n** = workflow engine (automates tasks)
- **HOJAI** = AI Workforce Platform (automates businesses)

The vision: Workflows → Skills → Employees → Departments → Companies → Networks

## 6. Core Node Types

### 1. Trigger Nodes
- Website Form, WhatsApp Message, Email Received, Calendar Event
- Payment Success, CRM Update, Sensor Event, Webhook
- Cron Job, Voice Command, Human Approval

### 2. Memory Nodes (Unique to HOJAI)
- Load Customer Memory, Update Preferences, Save Conversation
- Recall Past Purchases, Smart Forgetting, Context Injection

### 3. Twin Nodes (Unique to HOJAI)
- Create Customer Twin, Update Employee Twin, Load Organization Twin
- Query Relationship Graph, Generate Asset Twin

### 4. AI Agent Nodes
- Sales Agent, Support Agent, HR Agent, Finance Agent
- Marketing Agent, Founder Agent, Legal Agent, Procurement Agent

### 5. Intelligence Nodes
- Lead Score, Customer Lifetime Value, Churn Prediction
- Recommendation Engine, Fraud Detection, Demand Forecasting

### 6. SUTAR Nodes (Unique to HOJAI)
- Discover Suppliers, Negotiate Price, Generate Contract
- Verify Trust, Execute Purchase, Track Delivery

### 7. Human Nodes
- Approval Required, Review Contract, Assign Employee
- Request Feedback, Escalate Issue

## 11. HOJAI Workforce Architecture

```
Company
 ├── Departments
 │    ├── Employees
 │    │    ├── Skills
 │    │    │    ├── Workflows
 │    │    │    ├── Memory
 │    │    │    └── Tools
 │    │    └── Twins
 │    └── Policies
 └── Intelligence Layer
```

### Level 1: Atomic Skills
- Communication Skills: send_email, send_whatsapp, make_phone_call, schedule_meeting, etc.
- Sales Skills: qualify_lead, research_company, generate_proposal, etc.
- HR Skills: screen_resume, conduct_interview, generate_offer, etc.
- Finance Skills: process_invoice, detect_fraud, generate_report, etc.
- Procurement Skills: discover_supplier, compare_prices, negotiate_contract, etc.

### Level 2: Skill Packs
Multiple skills combined into packs (Sales Pack, HR Pack, etc.)

### Level 3: AI Employees
Skill packs combined with memory, twins, and permissions.

### Level 4: AI Departments
Multiple AI employees working together.

### Level 5: AI Companies
Full company structure with all departments.

## 13. The 7 Core SDKs

1. **@hojai/core** - Authentication, Events, Configuration, Runtime, Permissions, Multi-tenancy
2. **@hojai/flows** - Workflow engine equivalent to n8n
3. **@hojai/memory** - Conversation, Preference, Knowledge, Relationship, Learning Memory
4. **@hojai/twins** - Customer, Employee, Organization, Asset, Supplier, Partner Twins
5. **@hojai/agents** - Workforce layer with skills, employees, departments
6. **@hojai/connectors** - Universal integrations (300+ connectors)
7. **@hojai/sutar** - Trust Scores, Negotiation, Contracts, Procurement
