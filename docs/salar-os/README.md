## What is Salar OS?

**Salar OS** is the **AI Marketplace** within SUTAR OS - the premier destination for buying, selling, and subscribing to AI services, agents, and digital products across the RTMN ecosystem.

**Think of it as:**
- The "App Store" for AI
- The "Amazon" for AI services
- The "Stock Exchange" for AI agents
- The "Marketplace" where AI agents meet and transact

- **Create Listings** - List your AI services, agents, or products

## API Reference

### Base URL

```
Production: https://sutar.hojai.ai/api/marketplace
Staging:    https://sutar-staging.hojai.ai/api/marketplace
Local:      http://localhost:4250/api/marketplace
```

### Authentication

```http
Authorization: Bearer <jwt_token>
```

### 1. List Services

```http
GET /api/marketplace/services?category=ai-agents&limit=20&offset=0
```

**Query Parameters:**
- `category` - Filter by category
- `limit` - Number of results (default: 20, max: 100)
- `offset` - Pagination offset
- `sort` - Sort field (relevance, price, rating, popularity)
- `order` - asc or desc
- `minPrice` - Minimum price filter
- `maxPrice` - Maximum price filter
- `minRating` - Minimum rating filter
- `minTrustScore` - Minimum trust score filter

## Use Cases

### 1. Business Owner Buys AI Agent

```
Business Owner
    ↓
Searches "restaurant AI agent" on Salar OS
    ↓
Finds "HOJAI Restaurant AI Agent" (4.8 stars, 95 trust score)
    ↓
Compares pricing plans (Starter $29.99/mo, Pro $99.99/mo)
    ↓
Purchases Pro plan with credit card
    ↓
Receives API key and access URL
    ↓
Integrates AI agent into restaurant operations
    ↓
Leaves 5-star review after success
```

### 2. Developer Sells AI Service

```
Developer
    ↓
Creates AI service (e.g., translation agent)
    ↓
Lists on Salar OS with pricing and description
    ↓
Service goes through verification (KYC, quality check)
    ↓
Gets published with "Verified" badge
    ↓
Customers discover and purchase service
    ↓
Developer earns revenue (70% to developer, 30% to platform)
    ↓
Tracks sales via Provider Dashboard
    ↓
Receives monthly payouts
```

### 3. AI Agent Auto-Purchases Service

```
AI Agent A needs translation service
    ↓
Queries Salar OS marketplace via API
    ↓
Filters by category, price, trust score
    ↓
Selects best service based on criteria
    ↓
Initiates negotiation via Negotiation Engine
    ↓
Agrees on price ($0.001 per translation)
    ↓
Creates smart contract via Contract OS
    ↓
Purchases service via Economy OS
    ↓
Uses service immediately
    ↓
Trust Engine updates both parties' scores
```

### 4. Enterprise Bulk Purchase

```
Enterprise HR Department
    ↓
Needs 50 AI agents for different functions
    ↓
Searches Salar OS for HR agents
    ↓
Uses bulk purchase API
    ↓
Gets enterprise discount (30% off)
    ↓
Gets single invoice for all agents
    ↓
Deploys all agents across organization
    ↓
Tracks usage via Usage Tracker
    ↓
Renews subscription annually
```

## Integration Examples

### Example 1: Embed Salar OS in Your App

```html
<iframe 
  src="https://marketplace.salar.os/embed?category=ai-agents&theme=light"
  width="100%" 
  height="600px"
  frameborder="0">
</iframe>
```

### Example 2: Use Salar OS Widget

```html
<script src="https://cdn.salar.os/widget.js"></script>
<div id="salar-marketplace"></div>
<script>
  SalarWidget.init({
    containerId: 'salar-marketplace',
    apiKey: 'your_api_key',
    category: 'ai-agents',
    theme: 'auto'
  });
</script>
```

### Example 3: Programmatic Access

```javascript
import { SalarMarketplace } from '@hojai/salar-client';

const marketplace = new SalarMarketplace({
  apiKey: process.env.SALAR_API_KEY
});

// Search for services
const services = await marketplace.search({
  query: 'restaurant AI',
  filters: { category: 'ai-agents', minRating: 4.5 }
});

// Purchase a service
const purchase = await marketplace.purchase({
  serviceId: services[0].id,
  planId: 'plan_pro'
});

// Use the purchased service
const aiAgent = new HojaiAgent({
  apiKey: purchase.apiKey,
  endpoint: purchase.accessUrl
});

const result = await aiAgent.execute({
  task: 'optimize-menu-pricing',
  data: { /* ... */ }
});
```

### Example 4: List Your Own Service

```javascript
// Become a provider
const provider = await marketplace.registerProvider({
  name: 'My AI Company',
  email: 'me@myai.com',
  website: 'https://myai.com',
  description: 'We build amazing AI agents'
});

// Create a listing
const listing = await marketplace.createListing({
  providerId: provider.id,
  name: 'My Awesome AI Agent',
  category: 'ai-agents',
  subcategory: 'sales-agents',
  description: 'This AI agent does amazing things',
  pricing: {
    plans: [
      {
        name: 'Starter',
        price: 9.99,
        currency: 'USD',
        interval: 'monthly',
        features: ['100 API calls/day', 'Email support']
      },
      {
        name: 'Pro',
        price: 49.99,
        currency: 'USD',
        interval: 'monthly',
        features: ['Unlimited API calls', 'Priority support', 'Custom training']
      }
    ]
  },
  capabilities: ['nlp', 'vision', 'translation']
});

console.log('Listing created:', listing.id);
```
