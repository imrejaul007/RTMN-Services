## THE FIVE INTELLIGENCE LAYERS

### Layer 1: Reactive Intelligence
```
Rules, pattern matching, simple automation
Example: Fraud rules, keyword routing
```

### Layer 2: Predictive Intelligence
```
ML models, forecasting, anomaly detection
Example: Demand forecast, churn prediction, risk scoring
```

### Layer 3: Generative Intelligence
```
Content creation, reasoning, planning
Example: Campaign generation, contract drafting, strategy
```

### Layer 4: Autonomous Intelligence
```
Self-directed agents, multi-agent systems
Example: AI employees, autonomous negotiations
```

### Layer 5: Collective Intelligence
```
Network-wide knowledge, cross-entity learning
Example: Nexha discovery, trust graphs, market intelligence
```
