## Sales Process

### SMB (₹5,000-25,000/month)

1. **Demo Request** → Calendly booking
2. **Demo Call** (30 min) → Live API demo
3. **Trial** (7 days) → Free API access
4. **Convert** → Credit card checkout
5. **Onboard** → API docs + Slack support

### Mid-Market (₹25,000-2,00,000/month)

1. **Demo Request** → Sales call
2. **Discovery** (1 hour) → Understand needs
3. **Technical Demo** (1 hour) → API deep-dive
4. **Proof of Concept** (2 weeks) → Real integration
5. **Proposal** → Custom pricing
6. **Close** → Contract + MSA
7. **Onboard** → 30-day onboarding

### Enterprise (₹2,00,000+/month)

1. **Outreach** → LinkedIn/email
2. **Executive Briefing** (1 hour) → Vision + ROI
3. **Technical Workshop** (half-day) → Architecture
4. **Pilot** (30-60 days) → Real business impact
5. **Business Case** → ROI validation
6. **Procurement** → Legal + Finance
7. **Contract** → Annual + SLA
8. **Deploy** → 90-day onboarding
