## API Structure

```bash
# Base URL
https://hub.rtmn.in

# Core OS
/api/sales/*      → Sales CRM
/api/media/*      → Content
/api/marketing/*  → Campaigns

# Industry
/api/restaurant/* → Orders
/api/hotel/*      → Bookings
/api/healthcare/*  → Patients
/api/retail/*      → Products
/api/legal/*      → Cases

# Foundation
/api/identity/*   → CorpID
/api/memory/*     → MemoryOS
/api/twins/*      → TwinOS

# REZ
/api/crm/*        → Contacts
/api/wallet/*     → Payments
/api/care/*       → Support

# AdBazaar
/api/ads/*        → DSP
/api/audiences/*   → Segments
```
