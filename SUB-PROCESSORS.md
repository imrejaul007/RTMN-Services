# Sub-Processors

**Last Updated:** June 15, 2026

RTMN Group engages the following sub-processors to provide the Services. We notify customers 30 days before adding or replacing any sub-processor.

| Sub-processor | Purpose | Location | Data Processed |
|---------------|---------|----------|----------------|
| **Stripe** | Payment processing | USA, EU | Billing data, payment methods |
| **Amazon Web Services (AWS)** | Cloud hosting, compute, storage | USA, EU, APAC | All customer data (encrypted) |
| **Google Cloud Platform** | Backup, analytics | USA, EU | Service telemetry |
| **MongoDB Atlas** | Managed database | USA, EU, APAC | Customer data |
| **Redis Cloud** | Managed caching | USA, EU | Session data, cache |
| **Cloudflare** | CDN, DDoS protection, DNS | Global | HTTP traffic, static assets |
| **SendGrid (Twilio)** | Transactional email | USA | Email addresses, message content |
| **Postmark** | Critical transactional email | USA | Email addresses, message content |
| **Sentry** | Error tracking | USA | Error logs, breadcrumbs |
| **Intercom** | Customer support chat | USA, EU | Support conversations, account info |
| **Datadog** | Infrastructure monitoring | USA | Server metrics, logs |
| **OpenAI** | AI inference (optional, opt-in) | USA | Text prompts and completions (no training) |
| **Anthropic** | AI inference (optional, opt-in) | USA | Text prompts and completions (no training) |
| **Hugging Face** | Open-source model hosting | USA | Public model inference only |

## How to Receive Notifications

To receive notifications of sub-processor changes, subscribe at:
- Email: sub-processor-changes@rtmn.com
- RSS: [TBD]
- In-product: Settings → Compliance

## How to Object

If you have a reasonable objection to a new sub-processor, email legal@rtmn.com within 30 days of the change notification. We will work in good faith to find a resolution, which may include:
- Terminating the affected service
- Providing an alternative implementation
- Reverting the sub-processor change

## Compliance

All sub-processors are bound by written data processing agreements that impose obligations no less protective than those in our [DPA.md](DPA.md).

For the full list with addresses and certifications, contact: privacy@rtmn.com
