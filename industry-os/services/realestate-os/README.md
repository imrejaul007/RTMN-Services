# RealEstate OS

Industry-specific digital twin for real estate management.

## Port

**5230** - RealEstate OS

## API

- `GET/POST /api/properties` - Property management
- `GET/POST /api/listings` - Listing management
- `GET/POST /api/leads` - Lead management
- `GET/POST /api/agents` - Agent management
- `GET/POST /api/viewings` - Viewing scheduling
- `GET/POST /api/offers` - Offer management
- `GET /api/analytics` - Dashboard
