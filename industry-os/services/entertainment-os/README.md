# Entertainment OS

**Port:** 5200  
**Industry:** entertainment

Entertainment OS provides complete management for entertainment businesses.

## Quick Start

```bash
cd entertainment-os
npm install
npm start
```

## API Endpoints

### Authentication
- `POST /auth/register` - Register
- `POST /auth/login` - Login
- `GET /auth/verify` - Verify token
