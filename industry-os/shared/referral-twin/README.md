# Referral Twin

Digital twin service for referral program management.

## Port

**3016** - Referral Twin

## API

- `GET/POST /api/referrals` - Referral management
- `PATCH /api/referrals/:id/status` - Update status
- `GET/POST /api/programs` - Program management
- `GET/POST /api/rewards` - Reward tracking
- `GET /api/analytics` - Referral analytics
