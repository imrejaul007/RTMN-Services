# Agentic Mobility Startup Journey — MoveX AI

## TL;DR

A founder builds **MoveX AI** (agentic Uber) by:

1. Creating company in **HOJAI Studio** (30 minutes)
2. Hiring **AI workers from BAM** ($3,800/mo for 13 agents)
3. Launching **DO Mobility** app
4. Connecting to **Nexha** for insurance, fuel, service centers
5. Growing with **AI marketing agents**

**Result:** 10 humans + 200+ AI agents vs 10,000 humans at traditional Uber

## Phase 4: AI Agents Work Together

Ride Booking Flow:
1. Dispatch Agent receives request
2. Insurance Agent verifies coverage
3. Pricing Agent calculates fare
4. Dispatch Agent finds drivers
5. Payment Agent pre-authorizes
6. Safety Agent monitors trip
7. Finance Agent processes payout
