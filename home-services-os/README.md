# HomeServices OS

**Port:** 5140  
**Industry:** home_services

HomeServices OS provides complete management for home services businesses.

## Quick Start

```bash
cd home-services-os
npm install
npm start
```

## API Endpoints

### Authentication
- `POST /auth/register` - Register
- `POST /auth/login` - Login
- `GET /auth/verify` - Verify token
