## Integration API

### Layer Access Pattern

```javascript
// IndustryOS exposes unified API
const industryOS = {
  // Layer 1: Intelligence
  ai: {
    chat: (prompt) => genie.chat(prompt),
    copilot: (role) => businessCopilot[role](),
    agents: () => agentMarketplace.list(),
  },
  
  // Layer 2: Customer Growth
  customers: {
    list: () => crmHub.list(),
    add: (data) => crmHub.create(data),
    loyalty: (id) => loyalty.get(id),
    score: (id) => leadScoring.score(id),
  },
  
  // Layer 3: Commerce
  commerce: {
    procure: (items) => nexha.procure(items),
    rfq: (rfq) => nexha.createRFQ(rfq),
    franchise: () => franchiseOS.manage(),
  },
  
  // Layer 4: Financial
  finance: {
    wallet: () => wallet.balance(),
    pay: (to, amount) => wallet.transfer(to, amount),
    invoice: (data) => invoice.create(data),
  },
  
  // Layer 14: Autonomous
  autonomous: {
    goal: (objective) => goalOS.decompose(objective),
    decide: (context) => decisionEngine.evaluate(context),
    negotiate: (rfq) => negotiationEngine.start(rfq),
  },
};
```

### Unified SDK

```javascript
import { IndustryPlatform } from '@rtmn/sdk';

const restaurant = new IndustryPlatform({
  industry: 'restaurant',
  apiKey: 'xxx',
  layers: ['intelligence', 'commerce', 'finance', 'autonomous'], // Enable specific layers
});

// Access any layer
const ai = restaurant.intelligence.copilot('founder');
const response = await ai.ask("What's my top selling dish today?");

const result = await restaurant.commerce.procure({
  items: ['tomatoes', 'onions'],
  preferredSupplier: 'farm_123',
});
```
