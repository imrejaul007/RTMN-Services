## 🎓 Architecture Patterns

### Common Service Pattern
Each Genie service follows a consistent structure:
```
service-name/
├── CLAUDE.md              # Complete documentation
├── package.json           # Dependencies
└── src/
    ├── index.js          # Main Express server
    └── routes/
        ├── [domain1].js  # Domain-specific endpoints
        ├── [domain2].js
        └── ...
```

### Shared Conventions
- **Port Range:** 4716-4727 (12 services)
- **Framework:** Express.js with helmet/cors
- **Storage:** In-memory Maps for fast prototyping
- **Documentation:** Each service has comprehensive CLAUDE.md
- **Response Format:** `{ success: true, data: {...} }`
