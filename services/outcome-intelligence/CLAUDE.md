# Outcome Intelligence Service

**Port:** 4934  
**Status:** Production Ready  
**Description:** Tracks business impact of every ticket/interaction in the RTMN ecosystem.

---

## Overview

Outcome Intelligence is a dedicated service that quantifies the business value generated by customer support interactions, tickets, and service outcomes. It calculates revenue saved, customers retained, churn prevented, upsells generated, referrals created, and risks identified.

---

## Features

- **Revenue Saved Tracking** - Track revenue saved per ticket/interaction
- **Customer Retention** - Monitor customers retained through interventions
- **Churn Prevention** - Identify and track prevented customer churn
- **Upsell Generation** - Track upsell opportunities captured
- **Referral Creation** - Monitor referral generation from positive outcomes
- **Risk Identification** - Flag and categorize identified risks
- **CSAT Improvement** - Measure customer satisfaction before/after
- **Resolution Analytics** - Track resolution times and efficiency
- **Trend Analysis** - Period-over-period comparisons
- **Dashboard Views** - Real-time business impact dashboards
- **Report Generation** - Exportable reports with filters

```
POST /api/outcomes             # Track a new outcome
```

## Data Models

### Outcome Record
```typescript
{
  outcomeId: string;           // Unique ID (OUT-XXXXXXXX)
  tenantId: string;             // Tenant identifier
  ticketId: string;             // Source ticket ID
  interactionId?: string;       // Optional interaction ID
  timestamp: Date;              // When outcome was recorded
  revenueImpact: {
    saved: number;             // Revenue saved
    protected: number;         // Revenue protected
    cost: number;              // Cost of intervention
  };
  customerImpact: {
    retained: boolean;         // Customer retained
    churned: boolean;          // Customer churned
    promoted: boolean;         // Customer promoted
    npsBefore?: number;         // NPS before
    npsAfter?: number;          // NPS after
  };
  businessImpact: {
    upsell: boolean;           // Upsell generated
    upsellAmount?: number;     // Upsell value
    referral: boolean;         // Referral created
    referralCount?: number;    // Number of referrals
    riskIdentified: boolean;   // Risk identified
    riskSeverity?: 'low' | 'medium' | 'high' | 'critical';
  };
  metrics: {
    csatBefore?: number;       // CSAT before (1-5)
    csatAfter?: number;        // CSAT after (1-5)
    resolutionTimeMinutes: number;
    firstResponseTimeMinutes?: number;
    agentId?: string;
  };
  metadata?: Record<string, unknown>;
}
```

### Aggregated Metric
```typescript
{
  tenantId: string;
  period: 'daily' | 'weekly' | 'monthly';
  startDate: Date;
  endDate: Date;
  totals: {
    totalRevenueSaved: number;
    totalRevenueProtected: number;
    totalRevenueCost: number;
    totalCustomersRetained: number;
    totalCustomersChurned: number;
    totalUpsellsGenerated: number;
    totalUpsellAmount: number;
    totalReferralsCreated: number;
    totalRisksIdentified: number;
    totalOutcomesTracked: number;
  };
  averages: {
    avgCsatImprovement: number;
    avgResolutionTime: number;
    avgRevenuePerTicket: number;
  };
  trends: {
    revenueSavedTrend: number;
    retentionRate: number;
    upsellConversionRate: number;
  };
}
```

## Usage Examples

### Track an Outcome
```bash
curl -X POST http://localhost:4934/api/outcomes \
  -H "Content-Type: application/json" \
  -H "X-Tenant-ID: tenant-123" \
  -d '{
    "ticketId": "TKT-001",
    "revenueImpact": {
      "saved": 500,
      "protected": 1000,
      "cost": 50
    },
    "customerImpact": {
      "retained": true,
      "churned": false,
      "promoted": true,
      "npsBefore": 6,
      "npsAfter": 9
    },
    "businessImpact": {
      "upsell": true,
      "upsellAmount": 200,
      "referral": true,
      "referralCount": 2,
      "riskIdentified": false
    },
    "metrics": {
      "csatBefore": 3.5,
      "csatAfter": 4.8,
      "resolutionTimeMinutes": 45,
      "firstResponseTimeMinutes": 5,
      "agentId": "agent-001"
    }
  }'
```

### Get Analytics
```bash
curl http://localhost:4934/api/outcomes/analytics \
  -H "X-Tenant-ID: tenant-123"
```

### Get Dashboard
```bash
curl http://localhost:4934/api/outcomes/dashboard?period=30d \
  -H "X-Tenant-ID: tenant-123"
```

### Generate Report
```bash
curl "http://localhost:4934/api/outcomes/report?startDate=2026-01-01&endDate=2026-01-31&groupBy=agent" \
  -H "X-Tenant-ID: tenant-123"
```

## Service Integration

### Event Bus Integration
Subscribe to events from other services:
- `ticket.resolved` - When a ticket is resolved
- `customer.churned` - When customer churn detected
- `upsell.created` - When upsell opportunity captured
- `risk.detected` - When risk is identified

### Service Connections
| Service | Connection | Purpose |
|---------|------------|---------|
| REZ-ecosystem-connector | Port 4399 | Service registry |
| REZ-event-bus | Port 4510 | Pub/sub events |
| Memory OS | Port 4703 | Store outcome context |
