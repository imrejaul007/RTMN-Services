# Demo Script — BrandPulse

**Duration:** 5-7 minutes
**Audience:** Hotel/restaurant/retail decision-makers
**Goal:** Show real value in under 10 minutes

---

## Pre-Demo Checklist

- [ ] Screen share ready — close all unrelated tabs
- [ ] Demo environment is live (not localhost)
- [ ] Sample brand data is loaded (use "Demo Hotel Group" or "Sample Restaurant")
- [ ] Swagger UI is open in another tab (for API deep-dive)
- [ ] Timer is running

---

## Demo Flow

### 0:00 — Hook (30 seconds)

**Say:** "Every business knows they should be monitoring their reviews. But here's the reality — most are checking manually, once a week, on 1-2 sites. By the time they see a negative review, it's been public for days. BrandPulse changes that."

**Show:** Split screen — [Old way: spreadsheets, manual copy-paste] vs [BrandPulse: live dashboard]

---

### 0:30 — Dashboard Overview (1 minute)

**Say:** "This is the BrandPulse dashboard. One view, every review, every source, in real time."

**Walk through:**
1. **Brand selector** — Switch between brands
2. **Overall sentiment score** — Large number with trend arrow
3. **Review volume** — Today's new reviews vs average
4. **Source breakdown** — Google, Yelp, TripAdvisor, Facebook (pills with counts)
5. **Sentiment over time** — 30-day sparkline

**Demo tip:** Click on the sentiment score to show drill-down. Show 2 sources quickly.

---

### 1:30 — Real-Time Aggregation (1 minute)

**Say:** "BrandPulse pulls reviews from Google, Yelp, TripAdvisor, and Facebook automatically. Every new review appears here within minutes."

**Walk through:**
1. **Recent reviews feed** — Scroll through real examples
2. **Click a review** — Show full text + sentiment breakdown
3. **Click another source** — Show it's in the feed too

**Key point:** "No manual entry. No waiting. No spreadsheets."

---

### 2:30 — Sentiment Analysis (1.5 minutes)

**Say:** "BrandPulse doesn't just count reviews — it understands them. Every review is analyzed for overall sentiment and broken down by aspect."

**Walk through:**
1. **Sentiment distribution** — Positive / Neutral / Negative breakdown
2. **Click into a negative review** — Show aspect tags (e.g., "Service: -0.8", "Cleanliness: -0.3")
3. **Click into a positive review** — Show different aspects
4. **Aspect summary chart** — Bar chart showing which aspects score well/poorly

**Key point:** "You know you have a problem with 'service speed' — now you know exactly which reviews mention it."

---

### 4:00 — Alerts (1 minute)

**Say:** "The fastest way to kill a bad review is to respond quickly. BrandPulse alerts you the moment something needs attention."

**Walk through:**
1. **Alert rules** — Show "Negative review detected" rule
2. **Alert history** — Show recent alerts
3. **Webhooks** — "This alert can trigger a webhook to your CRM, Slack, or ticketing system"

**Key point:** "Your team knows within minutes, not days."

---

### 5:00 — API & Integration (1 minute)

**Say:** "BrandPulse is built for developers. Everything you see here is available via our REST API."

**Walk through:**
1. **Swagger UI** — Open API docs tab
2. **GET /api/v1/brands** — Show endpoint
3. **GET /api/v1/analytics/sentiment** — Show response structure
4. **Code snippet** — Show the JavaScript SDK call

**Key point:** "Integrate it into your existing tools, build your own dashboard, or automate workflows."

---

### 6:00 — RTNM Integration (1 minute)

**Say:** "BrandPulse is part of the RTNM ecosystem. Your hotel OS or restaurant OS can automatically pull in sentiment data."

**Walk through:**
1. **Show RTNM SDK integration example** — Code snippet
2. **Mention TwinOS Hub** — How sentiment becomes a customer twin attribute
3. **Mention GoalOS** — "Set a goal: 'Maintain 4.5+ sentiment on service' and get automated action plans"

**Key point:** "BrandPulse isn't just a standalone tool — it's intelligence that flows through your entire RTNM stack."

---

### 7:00 — Close (30 seconds)

**Say:** "BrandPulse gives you the visibility to understand what customers are saying, the speed to respond before damage is done, and the insights to improve over time."

**Show:** Quick recap of 3 key numbers (e.g., "28% increase in response rate, 34% improvement in sentiment, 3.2x ROI in 90 days")

**CTA:** "You can start with the free plan — one brand, 100 reviews/month, no credit card. Or I can set up a personalized demo with your actual data."
