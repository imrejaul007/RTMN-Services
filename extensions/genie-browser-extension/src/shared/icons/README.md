# Genie Browser Extension Icons

Place the following icon files in this directory:

- `icon16.png` (16x16) - Toolbar icon
- `icon48.png` (48x48) - Extension page icon
- `icon128.png` (128x128) - Chrome Web Store icon

You can use any image editor or online tool to create these icons. The recommended design is:

- Background: Gradient purple (#667eea to #764ba2)
- Symbol: 🧞 emoji or a stylized "G" letter

For a quick placeholder, you can create simple colored squares with the Genie purple gradient.
