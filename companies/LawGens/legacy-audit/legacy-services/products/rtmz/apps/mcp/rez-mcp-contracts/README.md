# REZ MCP Contracts

MCP server for contract management integration.

## Features

- Create and manage contracts
- E-signature workflows
- Document versioning
- Contract analytics

## Usage

```bash
npm install
npm run build
npm start
```

## Environment

```
PORT=3112
CONTRACT_URL=http://contract-mgmt:5003
AUTH_SERVICE_URL=http://rez-auth:4002
```
