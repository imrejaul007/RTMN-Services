# REZ MCP Legal

MCP server for legal document AI analysis.

## Features

- Legal document analysis
- Clause extraction
- Risk assessment
- Compliance checking

## Usage

```bash
npm install
npm run build
npm start
```

## Environment

```
PORT=3113
LEGAL_AI_URL=http://legal-ai:5004
AUTH_SERVICE_URL=http://rez-auth:4002
```
