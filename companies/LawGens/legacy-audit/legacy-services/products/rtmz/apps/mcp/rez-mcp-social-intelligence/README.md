# REZ MCP Social Intelligence

OSINT and social media intelligence MCP server.

## Tools
- `scrape_profile` - Scrape social media profiles
- `analyze_network` - Analyze contact networks
- `sentiment_analysis` - Content sentiment analysis
- `find_connections` - Find common connections
- `track_activity` - Track social activity

## Ports
- HTTP: 3130
- Health: `/health`

## Usage
```bash
npm install && npm run build && npm start
```
