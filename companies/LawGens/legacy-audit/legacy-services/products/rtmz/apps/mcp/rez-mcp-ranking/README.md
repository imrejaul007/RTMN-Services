# REZ MCP Ranking

MCP server for ML ranking service control.

## Features

- Submit ranking requests
- Monitor ranking scores
- A/B testing support
- Custom ranking models

## Usage

```bash
npm install
npm run build
npm start
```

## Environment

```
PORT=3115
RANKING_URL=http://ranking:5006
AUTH_SERVICE_URL=http://rez-auth:4002
```
