# REZ MCP Cosmic Twin

MCP server for Cosmic Twin digital twin control.

## Features

- Digital twin synchronization
- Real-time state management
- Predictive modeling
- Simulation controls

## Usage

```bash
npm install
npm run build
npm start
```

## Environment

```
PORT=3114
COSMIC_TWIN_URL=http://cosmic-twin:5005
AUTH_SERVICE_URL=http://rez-auth:4002
```
