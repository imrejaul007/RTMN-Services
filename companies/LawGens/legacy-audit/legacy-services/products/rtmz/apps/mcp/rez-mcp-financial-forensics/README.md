# REZ MCP Financial Forensics

Financial record analysis - invoice patterns, fraud detection, anomaly detection.

## Tools
- `analyze_invoices` - Analyze invoice patterns
- `detect_anomalies` - Detect unusual transactions
- `correlate_financial` - Link financial to communications
- `generate_financial_report` - Financial analysis report

## HTTP API
```bash
POST /analyze/invoices - Analyze invoices
POST /detect/anomalies - Detect anomalies
POST /correlate - Correlate with communications
GET /analysis/{id}/report - Get report
```

Port: 3130
