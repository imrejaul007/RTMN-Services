This is a **B2B commerce platform**. Businesses negotiate with each other (RFQ → quote → order → invoice → payment). Each business has SUTAR agents that handle procurement and sales autonomously.

## Architecture

- **backend:** node-express-typescript
- **SDKs:** @hojai/commerce, @hojai/sutar (supplier + buyer agents), @hojai/payment (invoice netting)

## SUTAR Agents

Two agent roles per business:
- **Supplier agent** — receives RFQs, generates quotes, negotiates terms
- **Buyer agent** — sends RFQs, evaluates quotes, places orders

- `POST /api/rfqs` — create RFQ

- `POST /api/orders` — buyer converts quote to order
- `POST /api/invoices` — invoice generation
