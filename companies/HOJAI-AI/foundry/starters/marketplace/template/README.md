# {{PROJECT_TITLE}}

> Marketplace starter built with **HOJAI Foundry** — generated by `npx hojai create`.

## Quick start

```bash
npm run dev
```

Then open:

- Frontend: <http://localhost:3000>
- Backend:  <http://localhost:4001>

## What's in here

- `apps/backend/` — Express + ESM backend on port 4001
- `apps/frontend/` — Static HTML+JS frontend on port 3000
- `scripts/dev.js` — Runs both servers in one process
- `hojai.ai.md` — AI-native spec (Claude Code / Cursor / Codex should read this first)
- `.hojai/manifest.json` — Machine-readable project schema
- `.hojai/capability.json` — CapabilityOS declaration for Nexha federation

## API at a glance

```bash
curl http://localhost:4001/health
curl http://localhost:4001/api/buyer/products
curl -X POST http://localhost:4001/api/buyer/rfqs \
  -H 'content-type: application/json' \
  -d '{"buyerId":"b1","productId":"<pid>","quantity":100}'
curl -X POST http://localhost:4001/api/agents/Sales \
  -H 'content-type: application/json' \
  -d '{"rfqId":"r1","productId":"<pid>","quantity":100}'
```
