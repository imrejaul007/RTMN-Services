## How to Extend

### Add a new SUTAR agent

1. Open `apps/backend/src/agents/index.js`
2. Add an entry to the `AGENTS` array:
   ```js
   { name: 'Marketing', description: '…', run: marketingRun }
   ```
3. Add the function below:
   ```js
   function marketingRun({ campaign }) { return { agent: 'Marketing', campaign }; }
   ```
4. Update `.hojai/capability.json` with the new capability.

### Replace the in-memory store with MongoDB

1. Add `mongoose` to dependencies
2. Replace `apps/backend/src/services/store.js` with a Mongoose-backed implementation
3. The function signatures stay the same — no other files need to change.

### Connect to the Global Nexha network

Run `npx hojai connect-nexha` (built in a future release) or POST to `/api/nexha/peers/connect` with your federation credentials.
