## How to Extend

### Add a new SUTAR agent

1. Open `apps/backend/src/agents/index.ts`
2. Add an entry to the `AGENTS` array
3. Wire the new agent's logic in the same file
4. Update `.hojai/capability.json` with the new capability

### Add a new endpoint

1. Create a file in `apps/backend/src/routes/`
2. Use the `requireAuth` middleware from `@hojai/foundation`
3. Validate inputs with Zod
4. Re-export the router from `apps/backend/src/index.ts`

### Connect to the Global Nexha network

POST to `/api/nexha/peers/connect` with your federation credentials, or run `npx hojai connect-nexha`.
