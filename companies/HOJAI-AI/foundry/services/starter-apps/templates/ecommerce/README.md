## 🚀 Quick Start

```bash
# 1. Generate the apps
curl -X POST http://localhost:4580/api/v1/generate \
  -H "Content-Type: application/json" \
  -d '{"companyName": "MyShop", "template": "ecommerce"}'

# 2. Extract and install
cd /path/to/generated/apps

# For Passenger App
cd passenger-app
npm install
npx expo start

# For Seller App  
cd seller-app
npm install
npx expo start

# For Admin Dashboard
cd admin-dashboard
npm install
npm start
```

## 🎨 Customization

### Branding
Edit `src/constants/theme.ts` to customize:
- Primary/Secondary colors
- Typography
- Spacing
- Border radius

### API Configuration
Update `src/config/index.ts`:
- API base URL
- Timeout settings
- Retry logic

### Features
Toggle features in `src/config/features.ts`:
- Enable/disable specific features
- Configure third-party integrations
