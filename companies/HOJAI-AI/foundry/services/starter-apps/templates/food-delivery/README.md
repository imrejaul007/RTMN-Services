## 📱 Apps Included

| App | Type | Description |
|-----|------|-------------|
| **Customer App** | Customer-facing | React Native mobile app for ordering food |
| **Seller App** | Restaurant-facing | React Native mobile app for restaurant partners |
| **Driver App** | Delivery-facing | React Native mobile app for delivery drivers |
| **Admin Dashboard** | Admin-facing | React web app for platform management |

## 🚀 Quick Start

```bash
# Generate the apps
curl -X POST http://localhost:4580/api/v1/generate \
  -H "Content-Type: application/json" \
  -d '{"companyName": "MyFoodApp", "template": "food-delivery"}'

# Customer App
cd customer-app && npm install && npx expo start

# Seller App
cd seller-app && npm install && npx expo start

# Driver App
cd driver-app && npm install && npx expo start

# Admin Dashboard
cd admin-dashboard && npm install && npm start
```
