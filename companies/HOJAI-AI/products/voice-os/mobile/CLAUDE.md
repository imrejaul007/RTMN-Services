# VoiceOS Mobile - React Native App

## Quick Start

```bash
cd products/voice-os/mobile
# Copy VoiceOSApp.ts to your Expo project
# Run: npx expo start
```

## Features

- Tap to speak
- Memory chips
- Quick actions
- TTS responses
- Dark theme

## API

Connects to:
- voice-orchestrator (:4898)
- memory-os (:4703)

## Build

```bash
npx expo run:ios
npx expo run:android
```
