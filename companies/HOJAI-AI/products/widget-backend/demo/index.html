📦 The HOJAI Widget

The chat bubble in the bottom-right corner is a single <script> tag that turns any website into an AI front office. It routes every visitor message to the right SUTAR agent — sales, support, booking, commerce — based on intent classification, enriched with real-time business intelligence.

⚡ Installation (5 seconds)

Add this snippet anywhere on your site:

<script src="https://cdn.hojai.ai/widget.js" data-key="pk_live_maya_..." data-company="maya-collective" data-color="#3B82F6" defer></script>

🧠 What's behind it

  • Intent Engine — classifies visitor messages (10+ intents)
  • REZ Intelligence — enriches with customer + merchant insights
  • SUTAR Agents — sales, support, booking, commerce agents
  • Memory Layer — remembers per-visitor context
  • TwinOS — updates customer twins with each interaction
