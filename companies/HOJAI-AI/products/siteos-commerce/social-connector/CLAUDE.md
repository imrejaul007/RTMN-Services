# HOJAI SiteOS Social Media Connector
**Port:** 5492

## Features
- Multi-platform (Facebook, Instagram, Twitter, LinkedIn, YouTube, TikTok)
- Post scheduling
- Content calendar
- Performance analytics

## API
- POST /api/posts
- GET /api/posts
- POST /api/schedule
- GET /api/calendar
- GET /api/metrics

## Start
```bash
npm install && npm start
```
