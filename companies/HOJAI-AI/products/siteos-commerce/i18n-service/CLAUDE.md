# HOJAI SiteOS i18n Service
**Port:** 5491

## Features
- 15+ locales (en, hi, ar, zh, es, fr, etc.)
- RTL support
- Namespace translations
- Key-value storage

## API
- GET /api/locales
- GET /api/translations/:locale
- POST /api/translations/:locale
- GET /api/rtl/:locale

## Start
```bash
npm install && npm start
```
