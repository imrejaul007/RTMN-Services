# HOJAI SiteOS SMS Service
**Port:** 5487

## Features
- Multi-provider (Twilio, MSG91, Gupshup)
- Bulk SMS
- DLT compliance (India)
- Delivery status tracking

## API
- POST /api/sms/send
- POST /api/sms/bulk
- GET /api/sms/stats
- GET /api/sms/status/:id

## Start
```bash
npm install && npm start
```
