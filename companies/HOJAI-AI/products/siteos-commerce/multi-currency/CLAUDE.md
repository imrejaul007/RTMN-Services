# HOJAI SiteOS Multi-Currency Service
**Port:** 5490

## Features
- 10+ currencies (USD, INR, EUR, GBP, AED, SAR, etc.)
- Real-time conversion
- Locale-based formatting
- RTL support detection

## API
- GET /api/currencies
- POST /api/convert
- POST /api/format
- GET /api/detect

## Start
```bash
npm install && npm start
```
