# HOJAI SiteOS Email Service
**Port:** 5486

## Features
- SMTP email sending (Nodemailer)
- Template management
- Bulk email campaigns
- Delivery tracking webhooks
- Personalization tokens

## API
- POST /api/email/send
- POST /api/email/bulk
- GET /api/email/stats
- GET /api/templates
- POST /api/templates

## Start
```bash
npm install && npm start
```
