# HOJAI SiteOS Push Notification Service
**Port:** 5488

## Features
- Web push notifications
- FCM integration
- Subscription management
- Segment-based push

## API
- POST /api/push/subscribe
- POST /api/push/send
- POST /api/push/send-segment
- GET /api/push/stats

## Start
```bash
npm install && npm start
```
