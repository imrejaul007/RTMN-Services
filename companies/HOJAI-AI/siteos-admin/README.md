# SiteOS Admin Dashboard

React dashboard for managing all 27 HOJAI SiteOS services.

## Start

```bash
cd products/siteos-admin
npm install
npm run dev
# Opens http://localhost:5173
```

## Pages

- Dashboard — Service health, stats, revenue
- Widgets — Configure widget appearance
- Analytics — Heatmaps, events, funnels
- Automation — Rules, templates
- Integrations — Connect tools
- Nexus — Product federation, agents
- Settings — API keys, service toggles
