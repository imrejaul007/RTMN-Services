# BLR Studio Integration

## Overview
Integrates BLR AI Marketplace with HOJAI Studio.

## Purpose
Bridges marketplace capabilities with Studio workflow.

## Key Features
- Studio integration
- Workflow bridging
- Deployment support
- Status tracking

## API Endpoints
- `GET /api/integrations` - List integrations
- `POST /api/integrations` - Create integration
- `GET /api/integrations/:id` - Get integration

## Startup
```bash
cd blr-ai-marketplace/services/bam-studio-integration && npm run dev
```
