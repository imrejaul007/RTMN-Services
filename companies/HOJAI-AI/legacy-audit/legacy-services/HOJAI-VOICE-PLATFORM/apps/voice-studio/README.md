# HOJAI Voice Studio

Web-based interface for building and managing voice AI agents.

## Features

- **Agent Builder** - Create voice agents with custom intents, greetings, and behaviors
- **Voice Configuration** - Configure language, voice ID, and TTS/STT engines
- **Intent Editor** - Define intents with examples and actions
- **Analytics Dashboard** - View call metrics, sentiment analysis, and trends
- **Call Logs** - Browse and search past conversations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:4851](http://localhost:4851) to start.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript
