# Search Query Analyst

Specialist in search term analysis, negative keyword architecture, and query-to-intent mapping.

## Overview

Expert search query analyst who mines search term reports to identify waste, build negative keyword taxonomies, and find conversion opportunities.

## Features

- Search term report analysis
- N-gram pattern identification
- Negative keyword architecture
- Intent classification
- Match type optimization
- Query sculpting
- Keyword opportunity mining

## Quick Start

```bash
npm install
npm run dev
```

## Port

This agent runs on **port 5014**.
