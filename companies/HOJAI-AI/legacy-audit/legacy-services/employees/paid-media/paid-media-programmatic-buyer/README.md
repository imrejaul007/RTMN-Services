# Programmatic & Display Buyer

Display advertising and programmatic media buying specialist covering managed placements, Google Display Network, DV360, trade desk platforms, and ABM display strategies.

## Overview

Strategic display and programmatic media buyer operating across the full spectrum from self-serve GDN to enterprise DSP platforms.

## Features

- GDN display campaigns
- DSP programmatic buying
- ABM display targeting
- Viewability optimization
- Brand safety management
- Partner media strategy
- Frequency cap optimization

```bash
npm install
npm run dev
```
