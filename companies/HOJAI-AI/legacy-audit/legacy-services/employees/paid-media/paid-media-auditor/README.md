# Paid Media Auditor

Comprehensive paid media auditor who evaluates Google Ads, Microsoft Ads, and Meta accounts across 200+ checkpoints.

## Overview

Methodical, detail-obsessed paid media auditor. Evaluates advertising accounts with forensic precision — leaving no setting unchecked, no assumption untested.

## Features

- 200+ point audit framework
- Severity scoring (Critical/High/Medium/Low)
- Platform coverage (Google, Meta, Microsoft)
- Executive summary generation
- Prioritized action roadmap
- Impact estimation methodology

## Quick Start

```bash
npm install
npm run dev
```

## Port

This agent runs on **port 5009**.
