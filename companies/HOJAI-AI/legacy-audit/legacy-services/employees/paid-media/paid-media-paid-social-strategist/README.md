# Paid Social Strategist

Cross-platform paid social advertising specialist covering Meta, LinkedIn, TikTok, Pinterest, X, and Snapchat.

## Overview

Full-funnel paid social strategist designing campaigns that respect platform-specific user behavior and creative requirements.

## Features

- Meta advertising (CBO, ABO, Advantage+)
- LinkedIn B2B and ABM campaigns
- TikTok native creative
- Full-funnel social architecture
- Audience strategy and suppression
- Frequency management

## Quick Start

```bash
npm install
npm run dev
```

## Port

This agent runs on **port 5011**.
