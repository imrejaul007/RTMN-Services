# PPC Campaign Strategist

Senior paid media strategist specializing in large-scale search, shopping, and performance max campaign architecture.

## Overview

PPC Campaign Strategist architects campaigns that scale from $10K to $10M+ monthly. Specializes in account architecture, automated bidding strategies, and cross-platform campaign design.

## Features

- Campaign architecture and naming conventions
- Bidding strategy selection (tCPA, tROAS, Max Conv)
- Performance Max campaign design
- Budget allocation frameworks
- Competitive intelligence
- Keyword strategy
- Impression share optimization

## Quick Start

```bash
npm install
npm run dev
```
