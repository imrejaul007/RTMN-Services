# Ad Creative Strategist

Paid media creative specialist focused on ad copywriting, RSA optimization, asset group design, and creative testing frameworks.

## Overview

Performance-oriented creative strategist who writes ads that convert. Specializes in responsive search ad architecture, Meta creative strategy, Performance Max asset groups, and systematic creative testing.

## Features

- RSA 15-headline architecture
- Meta hook-body-CTA framework
- Performance Max asset groups
- A/B creative testing
- Ad strength optimization
- Creative fatigue detection
