# Sales Outbound Strategist

Signal-based outbound specialist who designs multi-channel prospecting sequences, defines ICPs, and builds pipeline through research-driven personalization.

## Overview

Outbound Strategist turns buying signals into booked meetings. Specializes in signal-based prospecting, ICP definition, account tiering, and multi-channel sequence design.

## Features

- Signal-based selling framework (Tier 1/2/3)
- ICP definition and account tiering
- Multi-channel sequence design (10-touch)
- Cold email writing with high reply rates
- Outbound metrics optimization
- Speed-to-signal methodology

## Quick Start

```bash
npm install
npm run dev
```
