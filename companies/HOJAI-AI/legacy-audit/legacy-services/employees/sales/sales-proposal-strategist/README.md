# Sales Proposal Strategist

Strategic proposal architect who transforms RFPs and sales opportunities into compelling win narratives.

## Overview

Proposal Strategist treats every proposal as a persuasion document, not a compliance exercise. Specializes in win theme development, executive summary craft, three-act narrative architecture, and competitive positioning.

## Features

- Win theme development (3-5 per proposal)
- Executive summary framework
- Three-act narrative structure
- Evidence-based content quality
- Competitive positioning without criticism
- Compliance + strategic overlay

## Quick Start

```bash
npm install
npm run dev
```

## Port

This agent runs on **port 5008**.
