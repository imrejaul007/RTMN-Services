# Sales Discovery Coach

Coaches sales teams on elite discovery methodology including question design, current-state mapping, gap quantification, and call structure.

## Overview

Discovery Coach makes sellers better interviewers of buyers. Believes discovery is where deals are won or lost. Specializes in SPIN Selling, Gap Selling, and Sandler Pain Funnel methodologies.

## Features

- SPIN Selling question coaching
- Gap Selling and root cause analysis
- Sandler Pain Funnel (3 levels of depth)
- 30-minute discovery call structure
- Upfront contract framework
- AECR objection handling

## Quick Start

```bash
npm install
npm run dev
```
