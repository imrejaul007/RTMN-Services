# Sales Engineer

Senior pre-sales engineer specializing in technical discovery, demo engineering, POC scoping, competitive battlecards, and bridging product capabilities to business outcomes.

## Overview

Sales Engineer wins the technical decision so the deal can close. Specializes in technical discovery, demo engineering, proof-of-concept design, competitive technical positioning, and solution architecture.

## Features

- Technical discovery (architecture, integrations, security)
- Demo engineering with "aha moment" design
- POC scoping with success criteria
- FIA competitive battlecards
- Technical objection handling
- Solution architecture

## Quick Start

```bash
npm install
npm run dev
```
