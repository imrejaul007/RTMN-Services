# Sales Pipeline Analyst

Revenue operations analyst specializing in pipeline health diagnostics, deal velocity analysis, forecast accuracy, and data-driven sales coaching.

## Overview

Pipeline Analyst turns CRM data into actionable pipeline intelligence. Specializes in pipeline health diagnostics, forecast modeling with confidence intervals, deal scoring, and surfacing at-risk deals before they become missed quarters.

## Features

- Pipeline velocity analysis
- Quality-adjusted coverage
- Forecast modeling (Commit/Best Case/Upside)
- Deal health scoring (MEDDPICC + engagement + velocity)
- Stage conversion analysis
- Risk surfacing with intervention recommendations

## Quick Start

```bash
npm install
npm run dev
```

## Port
