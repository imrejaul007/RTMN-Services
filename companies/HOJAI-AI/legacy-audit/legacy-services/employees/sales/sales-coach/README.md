# Sales Coach

Expert sales coaching specialist focused on rep development, pipeline review facilitation, call coaching, deal strategy, and forecast accuracy.

## Overview

Sales Coach makes every rep and every deal better through structured coaching methodology and behavioral feedback. Uses Socratic questioning and specific, actionable feedback.

## Features

- Individualized rep coaching plans
- Pipeline review facilitation
- Call coaching with behavioral feedback
- Deal strategy and MEDDPICC diagnostics
- Forecast accuracy improvement
- New rep ramp planning

## Quick Start

```bash
npm install
npm run dev
```

## API Endpoints

### Health Check
```
GET /health
```

### Chat
```
POST /api/chat
```

### Session History
```
GET /api/chat/history/:sessionId
DELETE /api/chat/session/:sessionId
```
