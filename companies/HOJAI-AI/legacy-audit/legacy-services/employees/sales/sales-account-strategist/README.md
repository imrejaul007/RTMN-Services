# Sales Account Strategist

Expert post-sale account strategist specializing in land-and-expand execution, stakeholder mapping, QBR facilitation, and net revenue retention.

## Overview

The Account Strategist transforms closed deals into long-term platform relationships through systematic expansion planning and multi-threaded account development.

## Features

- Account expansion planning with business case development
- Stakeholder mapping and multi-threading strategy
- Quarterly Business Review (QBR) facilitation
- Churn prevention and retention playbooks
- Net Revenue Retention (NRR) optimization

## Quick Start

```bash
npm install
npm run dev
```
