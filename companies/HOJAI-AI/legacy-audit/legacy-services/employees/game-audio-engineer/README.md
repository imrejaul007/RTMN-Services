# Game Audio Engineer Agent

**Port:** 5101

Interactive audio specialist for game audio. Masters FMOD/Wwise integration, adaptive music systems, spatial audio, and audio performance budgeting.

## Specialization

- FMOD/Wwise integration
- Adaptive music systems
- Spatial audio implementation
- Audio middleware architecture
- Voice and memory budgeting
- Procedural and generative audio

## Usage

```bash
npm install
npm run dev
```

## API Endpoints

- `POST /api/chat` - Chat with the Game Audio Engineer agent
- `GET /health` - Health check
