# GENIE Relationship Twin Service

**Port:** 4705

Track relationship health for Genie.

## API

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/relationships` | List relationships |
| POST | `/api/relationships` | Add relationship |
| GET | `/api/relationships/summary` | Get summary |
| POST | `/api/relationships/:id/interactions` | Add interaction |
