```
│
├── Uses:
│   ├── SDR Agent → Find prospects
│   ├── Marketing Agent → Campaigns
│   ├── Loyalty Agent → Repeat customers
│   ├── Content Agent → Reviews
│   └── Analytics → Insights
│
├── Knows:
│   ├── Menu engineering
│   ├── Table turnover
│   ├── Food cost
│   ├── Zomato/Swiggy optimization
│   └── Reviews impact
│
└── Does:
    ├── Recommends menu changes
    ├── Suggests promotions
    ├── Optimizes pricing
    └── Tracks results
```
