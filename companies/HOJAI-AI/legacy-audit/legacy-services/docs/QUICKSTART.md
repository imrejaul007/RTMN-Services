# Quick Start

## Hojai AI - 5 Minute Setup

### 1. Install
```bash
npm install @hojai/whatsapp-ai
```

### 2. Configure
```bash
cp .env.example .env
```

### 3. Deploy
```bash
docker-compose up -d
```

### 4. Test
```bash
curl http://localhost:4570/health
```
