## WHAT HUMANS GET vs AUTOMATION GETS

### Human Mode (Interactive)

```
┌─────────────────────────────────────────────────┐
│  HotelOS Workspace                               │
├─────────────────────────────────────────────────┤
│                                                  │
│  Owner asks:                                     │
│  "Why did revenue drop this week?"             │
│                                                  │
│  BOA responds:                                  │
│  "Revenue dropped 12% because:                 │
│   • Weekend pricing was 10% above market       │
│   • 3 negative reviews (breakfast quality)     │
│   • Staff shortage caused slow service          │
│                                                  │
│  Recommended actions:                          │
│  [ ] Adjust weekend pricing (saves ₹50K/mo)   │
│  [ ] Address breakfast quality (recover 5 stars)│
│  [ ] Schedule 2 more staff (₹30K/mo)          │
│                                                  │
│  What would you like me to do?"               │
│                                                  │
└─────────────────────────────────────────────────┘
```

### Automation Mode (Background)

```
┌─────────────────────────────────────────────────┐
│  AUTOMATION RUNNING (No Human Input Needed)     │
├─────────────────────────────────────────────────┤
│                                                  │
│  System monitors 24/7:                         │
│                                                  │
│  • Stock low → Auto-reorder via Nexha         │
│  • Negative review → Auto-respond             │
│  • Payment due → Auto-remind via RABTUL     │
│  • Staff shift open → Auto-offer via CorpPerks│
│  • Demand spike → Auto-adjust pricing        │
│  • Customer inactive → Auto-retention offer   │
│                                                  │
│  Only alerts human for:                        │
│  • High-value decisions (₹1L+)               │
│  • Exceptions (fraud, compliance)              │
│  • Strategic recommendations                  │
│                                                  │
└─────────────────────────────────────────────────┘
```
