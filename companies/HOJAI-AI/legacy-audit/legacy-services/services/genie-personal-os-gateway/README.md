# Genie Personal Os-gateway

## Overview

This is the Genie personal os-gateway service for HOJAI Genie AI.

**Tagline:** "You don't use Genie. You talk to Genie."

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker build -t genie-personal-os-gateway .
docker run -p 3000:3000 genie-personal-os-gateway
```
