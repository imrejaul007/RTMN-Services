# HOJAI Patient App

A mobile application for patients to manage their healthcare needs.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Run on iOS/Android:
   ```bash
   npm run ios
   npm run android
   ```

## Features

- Book and manage appointments
- View prescriptions
- Health records
- Chat with clinic
- Appointment reminders
- Profile management

## Tech Stack

- Expo Router
- React Native
- TypeScript
- Tailwind CSS (NativeWind)
- Zustand
