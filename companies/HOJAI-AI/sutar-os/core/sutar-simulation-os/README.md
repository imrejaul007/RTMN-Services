# SUTAR Simulation OS

## Purpose
Simulation environment for testing agent behaviors.

## Key Features
- Scenario simulation
- Outcome prediction
- Risk analysis
- What-if scenarios

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /health | Health check |
| POST | /api/simulate | Run simulation |
| GET | /api/simulations | List simulations |
| GET | /api/simulations/:id | Get simulation |
