# SUTAR Network Learning

## Purpose
Agent network learning and knowledge sharing.

## Key Features
- Knowledge propagation
- Pattern learning
- Network synchronization
- Learning analytics

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /health | Health check |
| POST | /api/learn | Record learning |
| GET | /api/learnings | Get learnings |
| POST | /api/sync | Sync network |
