# SUTAR Agent Network

## Purpose
Agent networking, discovery, and inter-agent communication.

## Key Features
- Agent discovery
- Network topology
- Agent relationships
- Communication routing

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /health | Health check |
| GET | /api/agents | List agents |
| GET | /api/agents/:id | Get agent |
| POST | /api/connect | Connect agents |
| GET | /api/network | Get network topology |
