# SUTAR Usage Tracker

## Purpose
Track and report agent usage metrics and costs.

## Key Features
- Usage recording
- Cost calculation
- Usage analytics
- Budget tracking

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /health | Health check |
| POST | /api/usage | Record usage |
| GET | /api/usage | Get usage stats |
| GET | /api/costs | Get cost summary |
