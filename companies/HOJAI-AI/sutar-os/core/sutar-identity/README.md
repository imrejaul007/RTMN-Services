# SUTAR Identity

## Purpose
Universal identity management for agents and entities in the SUTAR ecosystem.

## Key Features
- Identity creation and management
- Role-based access control
- Identity verification
- Delegation support

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | /health | Health check |
| POST | /api/identities | Create identity |
| GET | /api/identities/:id | Get identity |
| PUT | /api/identities/:id | Update identity |
| DELETE | /api/identities/:id | Delete identity |
