> **Company:** KHAIRMOVE Logistics (RTMN Group)
> **Industry:** Logistics & Freight Forwarding
> **SUTAR Services Used:** Nexha Discovery, Negotiation Engine, Contract OS, Trust Engine
> **Timeline:** June 2026

---

## The Challenge

KHAIRMOVE Logistics processes 500+ freight quotes monthly across 40+ shipping lanes. Manual negotiation was:

- **Slow:** 3-7 days per quote cycle
- **Expensive:** Dedicated team of 8 freight brokers costing $960K/year
- **Inconsistent:** Different brokers got different rates from same carriers
- **Reactive:** Couldn't proactively optimize routes or volumes

---

## What KHAIRMOVE Built

KHAIRMOVE deployed **KHAIRMOVE SUTAR** — an autonomous freight agent that:

1. **Listens** to inbound shipment requests via Nexha Federation
2. **Discovers** optimal carriers via Nexha Discovery (filters by reliability, price, capacity)
3. **Negotiates** multi-party with carriers via ACP Protocol
4. **Executes** smart contracts via Contract OS
5. **Tracks** shipments via IoT integration and updates all parties

## How It Works

### Step 1: Shipment Request Arrives
```
Importer ERP ──► Nexha Federation ──► KHAIRMOVE SUTAR
{
  "origin": "Kaohsiung, Taiwan",
  "destination": "Mumbai, India",
  "cargo_type": "electronics",
  "volume": "3×40ft containers",
  "weight": "42 metric tons",
  "incoterms": "CIF",
  "urgency": "standard",
  "budget": 12000
}
```

### Step 2: Carrier Discovery
KHAIRMOVE SUTAR queries Nexha Discovery:
- 12 carriers found active on Taiwan-India lane
- Filtered by: reliability score > 85, capacity available, rate competitiveness
- Top 5 carriers shortlisted in **3 seconds**

### Step 3: Multi-Carrier Negotiation
```
KHAIRMOVE SUTAR ──► ACP QUERY ──► Carrier A
                                    ├──► Carrier B
                                    ├──► Carrier C
                                    ├──► Carrier D
                                    └──► Carrier E

Each carrier responds with initial quote via ACP QUOTE.

KHAIRMOVE SUTAR runs 6 rounds of counter-offer with top 3 carriers,
optimizing for: price, transit time, reliability, capacity fit.

Final round: Best offer selected, others notified of rejection.
```

### Step 4: Contract Execution
- Smart contract generated by Contract OS
- All terms: freight rate, surcharges, demurrage, free time, penalties
- Escrow deposited (5% of freight value)
- Digital signatures via ACP Protocol

### Step 5: Shipment Tracking
- IoT sensors feed GPS + temperature data to KHAIRMOVE SUTAR
- Any delay > 2 hours → SUTAR auto-negotiates with alternative carriers
- Customer receives WhatsApp updates via Genie AI
