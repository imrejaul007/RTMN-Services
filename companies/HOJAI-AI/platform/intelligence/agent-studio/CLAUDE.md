# Agent Studio

**Port:** 4189
**Type:** Platform Intelligence Service (Agent Debugger)
**Package:** `@hojai/agent-studio`

## Overview

Debugger for agent executions. Traces each step, captures inputs/outputs/tokens/latency, replays sessions, and supports break-on-condition. Pairs with Agent Builder (4188) for blueprint authoring and Multi-Agent Runtime (4190) for execution traces.
