# Multi-Agent Runtime

**Port:** 4190
**Type:** Platform Intelligence Service (Agent Orchestration)
**Package:** `@hojai/multi-agent-runtime`

## Overview

Formalizes multi-agent patterns: sequential, parallel, pipeline, fan-out, fan-in, conditional. Orchestrates agent collaboration with shared state, retry policies, and timeout handling. Receives blueprints from Agent Builder (4188) and emits traces to Agent Studio (4189).
