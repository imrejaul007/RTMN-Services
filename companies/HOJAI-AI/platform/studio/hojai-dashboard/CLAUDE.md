# HOJAI Dashboard

React + Tailwind dashboard for managing HOJAI services.

## Quick Start

```bash
cd studio/hojai-dashboard
npm install
npm run dev
```

## Pages

- **Dashboard** - Overview stats, quick actions, services status
- **Connectors** - Configure Twilio, WhatsApp, Background Check
- **Services** - Reply Drafting, Refund Approval, RCA, ROI Calculator
- **Integrations** - HubSpot, Salesforce, Slack, Gmail
- **Analytics** - Performance metrics
- **Settings** - API keys, notifications

## API Integration

Connects to HOJAI API Gateway at `http://localhost:4500`.

## Build

```bash
npm run build  # Production build
```
