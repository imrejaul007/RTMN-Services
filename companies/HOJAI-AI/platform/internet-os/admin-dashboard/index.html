API Documentation

Base URL

http://localhost:4595

Endpoints

Health

  • GET /health - Service health
  • GET /ready - Readiness check
  • GET /api/stats - Statistics

Actors

  • GET /api/actors - List all 17 actors
  • GET /api/actors/:id - Get actor info
  • POST /api/actors/:id/run - Run actor (body: action, params)
  • POST /api/actors/batch - Batch run (body: inputs array)
  • GET /api/actors/search/:query - Search actors

Watchers

  • GET /api/watchers - List watchers
  • POST /api/watchers - Create watcher
  • GET /api/watchers/:id - Get watcher
  • GET /api/watchers/:id/state - Get state
  • GET /api/watchers/:id/changes - Get changes
  • POST /api/watchers/:id/start - Start
  • POST /api/watchers/:id/stop - Stop
  • DELETE /api/watchers/:id - Delete

Research

  • GET /api/research/agents - List agents
  • POST /api/research/market - Market research
  • POST /api/research/competitor - Competitor research
  • POST /api/research/procurement - Procurement research
  • POST /api/research/run-all - Run all
  • GET /api/research/reports - List reports
  • GET /api/research/stats - Stats

Scheduler

  • GET /api/scheduler/ - List schedules
  • POST /api/scheduler/ - Create schedule
  • POST /api/scheduler/:id/run - Run now
  • POST /api/scheduler/:id/enable - Enable
  • POST /api/scheduler/:id/disable - Disable
  • POST /api/scheduler/start - Start all
  • POST /api/scheduler/stop - Stop all
  • GET /api/scheduler/stats - Stats

History

  • GET /api/history - Search history
  • GET /api/history/entity/:id - Entity history
  • GET /api/history/stats - Stats
