# Lead Twin - Port 4894

## Overview
Lead scoring, qualification, activity tracking.

## Key Features
- Lead scoring
- Qualification tracking
- Activity history
- Conversion prediction

## API Endpoints
- GET /api/leads - List leads
- POST /api/leads - Create lead
- GET /api/leads/:id/score - Get score

## Environment
- Port: 4894

## Startup
cd platform/twins/lead-twin && npm run dev
