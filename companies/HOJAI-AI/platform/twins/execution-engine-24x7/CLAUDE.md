# Execution Engine 24x7

## Overview
Continuous task execution outside working hours.

## Key Features
- Task scheduling
- Autonomous execution
- Human oversight
- Exception handling

## API Endpoints
- GET /api/tasks - List tasks
- POST /api/tasks - Create task
- POST /api/execute/:taskId - Execute task

## Startup
cd platform/twins/execution-engine-24x7 && npm run dev
