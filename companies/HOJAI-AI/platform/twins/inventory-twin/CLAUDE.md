# Inventory Twin

## Overview
Inventory levels, stock predictions, reorder points.

## Key Features
- Stock levels
- Reorder prediction
- Demand forecasting
- Supplier integration

## API Endpoints
- GET /api/inventory - List inventory
- POST /api/inventory - Add item
- GET /api/inventory/:id/reorder - Get reorder

## Startup
cd platform/twins/inventory-twin && npm run dev
