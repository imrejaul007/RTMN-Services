# Reputation Twin

## Overview
Reputation scoring, reviews, trust metrics.

## Key Features
- Review aggregation
- Trust scoring
- Sentiment analysis
- Reputation trends

## API Endpoints
- GET /api/reputation/:entityId - Get reputation
- POST /api/reviews - Add review
- GET /api/trust/:entityId - Get trust

## Startup
cd platform/twins/reputation-twin && npm run dev
