# Skill Twin

## Overview
Skills, capabilities, certifications.

## Purpose
Tracks individual skills and capabilities.

## Key Features
- Skill inventory
- Certification tracking
- Skill gaps
- Learning paths

## API Endpoints

### Skills
- `GET /api/skills` - List skills
- `POST /api/skills` - Add skill
- `GET /api/skills/:id` - Get skill

### Certifications
- `GET /api/certifications` - List certifications

## Startup
```bash
cd platform/twins/skill-twin && npm run dev
```
