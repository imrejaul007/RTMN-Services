# Organization Twin - Port 4710

## Overview
Company/org structure, hierarchy, KPIs.

## Key Features
- Org chart
- Hierarchy tracking
- KPI monitoring
- Structure analysis

## API Endpoints
- GET /api/org - Get organization
- GET /api/departments - List departments
- GET /api/kpis - Get KPIs

## Environment
- Port: 4710

## Startup
cd platform/twins/organization-twin && npm run dev
