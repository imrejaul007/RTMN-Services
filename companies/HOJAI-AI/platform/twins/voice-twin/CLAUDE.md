# Voice Twin - Port 4876

## Overview
Voice profiles, speech patterns, audio identity.

## Key Features
- Voice profile
- Speech patterns
- Audio identity
- Voice synthesis

## API Endpoints
- GET /api/voice/:userId - Get voice profile
- POST /api/voice - Create profile
- POST /api/synthesize - Synthesize speech

## Environment
- Port: 4876

## Startup
cd platform/twins/voice-twin && npm run dev
