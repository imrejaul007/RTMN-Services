# Order Twin - Port 4885

## Overview
Order lifecycle, fulfillment, returns tracking.

## Key Features
- Order tracking
- Fulfillment status
- Return handling
- Delivery prediction

## API Endpoints
- GET /api/orders - List orders
- POST /api/orders - Create order
- GET /api/orders/:id/tracking - Get tracking

## Environment
- Port: 4885

## Startup
cd platform/twins/order-twin && npm run dev
