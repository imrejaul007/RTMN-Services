# Product Twin - Port 4720

## Overview
Product catalog, specifications, pricing.

## Key Features
- Product catalog
- Specifications
- Pricing history
- Demand patterns

## API Endpoints
- GET /api/products - List products
- POST /api/products - Create product
- GET /api/products/:id/specs - Get specs

## Environment
- Port: 4720

## Startup
cd platform/twins/product-twin && npm run dev
