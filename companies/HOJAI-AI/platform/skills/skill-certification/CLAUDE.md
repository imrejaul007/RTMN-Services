# Skill Certification

## Overview
Certification system for skill quality.

## Purpose
Certifies and verifies skill quality.

## Key Features
- Quality review
- Certification levels
- Compliance check
- Badges and trust

## API Endpoints
- `GET /api/certifications` - List certifications
- `POST /api/certifications` - Submit for cert
- `GET /api/certifications/:id` - Get status

## Startup
```bash
cd platform/skills/skill-certification && npm run dev
```
