# Enterprise Skill Portal

## Overview
Self-service portal for enterprise skill management.

## Purpose
UI for enterprises to browse and install skills.

## Key Features
- Skill catalog
- Team management
- License management
- Analytics dashboard

## API Endpoints
- `GET /api/portal/skills` - Browse skills
- `POST /api/portal/teams` - Manage teams
- `GET /api/portal/analytics` - Usage analytics

## Startup
```bash
cd platform/skills/enterprise-skill-portal && npm run dev
```
