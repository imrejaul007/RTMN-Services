# Creator Payout

## Overview
Revenue sharing and payouts for skill creators.

## Purpose
Manages payments to skill creators.

## Key Features
- Revenue tracking
- Payout calculations
- Payment processing
- Transaction history

## API Endpoints
- `GET /api/payouts` - List payouts
- `POST /api/payouts` - Initiate payout
- `GET /api/revenue/:creatorId` - Get revenue

## Startup
```bash
cd platform/skills/creator-payout && npm run dev
```
