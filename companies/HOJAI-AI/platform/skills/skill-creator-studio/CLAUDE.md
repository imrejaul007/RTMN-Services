# Skill Creator Studio

## Overview
IDE for building AI skills.

## Purpose
Visual builder for skill development.

## Key Features
- Visual editor
- Testing environment
- Version control
- Publishing wizard

## API Endpoints
- `GET /api/studio/projects` - List projects
- `POST /api/studio/projects` - Create project
- `POST /api/studio/publish` - Publish skill

## Startup
```bash
cd platform/skills/skill-creator-studio && npm run dev
```
