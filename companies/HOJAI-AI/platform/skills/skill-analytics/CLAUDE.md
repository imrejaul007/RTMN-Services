# Skill Analytics

## Overview
Analytics and insights for skill usage.

## Purpose
Track skill performance and ROI.

## Key Features
- Usage metrics
- Performance dashboards
- ROI tracking
- Trend analysis

## API Endpoints
- `GET /api/analytics` - Get analytics
- `GET /api/analytics/:skillId` - Skill metrics
- `GET /api/analytics/dashboard` - Dashboard data

## Startup
```bash
cd platform/skills/skill-analytics && npm run dev
```
