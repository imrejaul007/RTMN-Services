# Industry Packs

## Overview
Pre-built skill packs for specific industries.

## Purpose
Industry-specific skill bundles for rapid deployment.

## Key Features
- Industry templates
- Role-based bundles
- Compliance packs
- Quick setup

## API Endpoints
- `GET /api/packs` - List industry packs
- `GET /api/packs/:id` - Get pack details
- `POST /api/packs/:id/install` - Install pack

## Startup
```bash
cd platform/skills/industry-packs && npm run dev
```
