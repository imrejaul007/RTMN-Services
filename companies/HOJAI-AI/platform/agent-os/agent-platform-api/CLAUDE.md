# Agent Platform API

## Overview
Gateway for AgentOS services.

## Purpose
Unified entry point.

## Key Features
- Health aggregation
- Service routing
- API gateway
- Rate limiting

## API Endpoints
- `GET /api/agent/platform/status` - Platform status
- `GET /api/agent/platform/health` - Health check

## Startup
```bash
cd platform/agent-os/agent-platform-api && npm run dev
```
