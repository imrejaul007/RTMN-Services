# Agent Execution Engine

## Overview
Execute agent tasks and plans.

## Purpose
Runs agent tasks reliably.

## Key Features
- Task execution
- Plan execution
- Result tracking
- Error recovery

## API Endpoints
- `POST /api/execute` - Execute task
- `GET /api/executions/:id` - Get execution
- `POST /api/execute/plan` - Execute plan

## Startup
```bash
cd platform/agent-os/agent-execution-engine && npm run dev
```
