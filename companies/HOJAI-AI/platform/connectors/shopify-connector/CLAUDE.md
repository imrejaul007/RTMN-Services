# Shopify Connector

## Overview
E-commerce integration for TwinOS.

## Purpose
Syncs Shopify store data with merchant twins.

## Key Features
- Order sync
- Product mapping
- Customer tracking
- Inventory sync

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/shopify-connector && npm run dev
```
