# Linear Connector

## Overview
Project management integration for TwinOS.

## Purpose
Syncs Linear issues with employee twins.

## Key Features
- Issue sync
- Cycle tracking
- Team mapping
- Workflow sync

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/linear-connector && npm run dev
```
