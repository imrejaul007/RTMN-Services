# Freshdesk Connector

## Overview
Customer support integration for TwinOS.

## Purpose
Syncs support tickets with customer twins.

## Key Features
- Ticket sync
- Customer mapping
- Resolution tracking
- SLA monitoring

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/freshdesk-connector && npm run dev
```
