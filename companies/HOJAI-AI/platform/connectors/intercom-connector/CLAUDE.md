# Intercom Connector

## Overview
Customer messaging integration for TwinOS.

## Purpose
Syncs Intercom conversations with customer twins.

## Key Features
- Conversation sync
- Customer mapping
- Message tracking
- Tag sync

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/intercom-connector && npm run dev
```
