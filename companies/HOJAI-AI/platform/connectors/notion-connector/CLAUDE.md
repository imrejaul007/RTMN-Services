# Notion Connector

## Overview
Knowledge base integration for TwinOS.

## Purpose
Syncs Notion pages with knowledge twins.

## Key Features
- Page sync
- Database mapping
- Comment tracking
- Permission sync

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/notion-connector && npm run dev
```
