# Asana Connector

## Overview
Project management integration for TwinOS.

## Purpose
Syncs Asana tasks and projects with employee twins.

## Key Features
- Task sync
- Project sync
- Team mapping
- Due date tracking

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/asana-connector && npm run dev
```
