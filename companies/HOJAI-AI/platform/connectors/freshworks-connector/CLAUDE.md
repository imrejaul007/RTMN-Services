# Freshworks Connector

## Overview
CRM integration for TwinOS.

## Purpose
Syncs Freshworks CRM data with customer twins.

## Key Features
- Contact sync
- Deal tracking
- Activity logging
- Pipeline sync

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/freshworks-connector && npm run dev
```
