# HubSpot Connector

## Overview
CRM integration for TwinOS.

## Purpose
Syncs HubSpot CRM with customer twins.

## Key Features
- Contact sync
- Deal tracking
- Activity logging
- Pipeline management

## API Endpoints
- `GET /connect` - Test connection
- `POST /sync` - Trigger sync

## Startup
```bash
cd platform/connectors/hubspot-connector && npm run dev
```
