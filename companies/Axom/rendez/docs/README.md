# Rendez — Documentation Index

Rendez is an exclusive REZ-ecosystem dating app. Real identity (REZ SSO), real intent (gifts), real meetups (QR check-in + REZ coin rewards).

## Repositories

| Repo | Description |
|------|-------------|
| `rendez-backend/` | Node.js + TypeScript REST + WebSocket API |
| `rendez-app/` | React Native (Expo) iOS + Android app |
| `rendez-admin/` | Next.js admin dashboard |
